You are a senior product-focused full-stack engineer.

Build a MOBILE-FIRST POC web application called:

“Influencer Deal, Contract & Billing Platform”

Primary users:
- Influencers (main users)
- Brands (interact via influencers)

The app is mainly used on mobile phones.
UI MUST be mobile-first, clean, modern, and touch-friendly.
Design it like a SaaS mobile app so it can later be converted into a mobile app.

------------------------------------
TECH STACK
------------------------------------
Frontend:
- React
- Tailwind CSS (mobile-first)

Backend:
- Node.js + Express

Storage:
- In-memory or JSON file storage (POC only)

File Upload:
- Local file storage for PDFs / images

Payments:
- Mock payment flow (no real gateway)

Auth:
- Mock login (single influencer)

------------------------------------
CORE PRODUCT IDEA
------------------------------------
Influencers manage all brand deals professionally.
Influencers tell brands:
“I use this platform to manage contracts, deliverables & billing.”

No monthly subscription.
Monetization happens PER CONTRACT only.

------------------------------------
MONETIZATION MODEL (STRICT)
------------------------------------
For every brand contract:
- Contract Creation Fee: ₹499
- Platform / Deliverables Management Fee: ₹500

Total platform charge per contract = ₹1000

No subscriptions.
No ads.
No other monetization.
